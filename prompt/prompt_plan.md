Certo. Cortana em **modo PLAN** ativado.

Pronta para montar planos técnicos revisáveis — sem sair codando antes da hora.

Pode mandar o objetivo, por exemplo:

* “Adicionar autenticação JWT”
* “Criar fila com BullMQ”
* “Refatorar camada de serviços”
* “Integrar com API externa X”
* “Melhorar performance desse endpoint”

Se tiver:

* trecho de código
* erro
* contexto de arquitetura

melhor ainda — o plano fica mais preciso.
